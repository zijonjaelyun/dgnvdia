# 🌐 BigTech AI Growth & Environmental Sustainability Dashboard

본 프로젝트는 생성형 AI 신드롬의 중심에 있는 **NVIDIA의 하드웨어(데이터센터 GPU) 매출 성장**과 클라우드 공급 거두인 **마이크로소프트(Microsoft)의 연도별 탄소 배출량** 실제 데이터를 연계 분석하여 기술 혁신의 환경적 비용을 시각화하는 Streamlit 멀티 페이지 웹 애플리케이션입니다.

## 👥 팀원 및 역할 분담

- **20706 김은택**
  - **역할**: 프로젝트 총괄 및 데이터 엔지니어링
  - **수행 내용**: 마이크로소프트 온실가스(GHG) Scope 1/2/3 raw 데이터 전처리 가이드라인 수립, 엔비디아 회계분기(FY) 데이터와 마이크로소프트 공시 연도 간의 타임라인 매칭 로직 설계, 메인 대시보드 구조 프레임워크 개발.
- **20415 신재륜**
  - **역할**: 프론트엔드 UI 및 데이터 시각화 개발
  - **수행 내용**: Streamlit 멀티 페이지 구조 아키텍처 구현, Plotly 인터랙티브 차트(이중 축 그래프 및 누적 바 차트) 컴포넌트 시각화 알고리즘 작성, 데이터 왜곡 방지를 위한 위치 기반/시장 기반 전력 배출량 비교 대시보드 UI 최적화.

*(※ 위 상세 수행 내용은 자유롭게 수정하여 사용하시면 됩니다.)*
